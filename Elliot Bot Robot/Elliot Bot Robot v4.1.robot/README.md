# **Elliot Bot Robot**

**Elliot Bot Robot** - `Новое поколение Elliot Bot`, которое я переписал и добавил много фишек

## Что нового добавлено в **Elliot Bot Robot 4.1.robot**

### Функции

#### Обновлённый `Терминал - Функция 10` `(Elliot Terminal)`


| Команда (Новые) | Значение |
|---------|----------|
| `/dir` | Список всех директорий |
| `/run` | Запуск программы |
| `/cfile` | Создать файл |
| `/wfile` | Записать в файл |
| `/afile` | Дописать в файл |
| `/rfile` | Прочитать файл |
| `/dfile` | Удалить файл |
| `/cdir` | Создать папку |
| `/deldir <-rm/-rmtree>` | Удалить папку (пустую или со всем содержимым) |
| `/enotes` | Зашифрованные заметки (Elliot Notes) |
| `/environment` | Информация об окружении |
| `/os` | Операционная система |
| `/count <root/user/ghost>` | Временно стать администратором / пользователем / призраком |
| `/mai` | Модель ИИ |
| `/encode` | Кодировка системы |
| `/status <root/user/ghost>` | Проверить, являюсь ли root/user/ghost |
| `/ips` | Локальный IP-адрес (через сокеты) |
| `/ipp` | Публичный IP-адрес |
| `/ippm` | Метаданные публичного IP (геолокация, провайдер) |
| `/ping` | Пинг до сайта или IP |
| `/whois` | Информация о домене или IP |
| `/obrowser` | Открыть веб-браузер |
| `/edir` | Проверить, существует ли путь |
| `/nstat` | Активные сетевые подключения |
| `/!exit` | Полное завершение программы |
| `/!aexit` | Выход из аккаунта |
| `/ncount <root/user/ghost>` | Перестать быть root/user/ghost |
| `/screen` | Сделать скриншот |
| `/ai` | Запуск AI-чата (Elliot Bot Robot AI) |
| `/random` | Случайные эффекты / вирусы-шутки |
| `/secret` | Секретная команда (выключение ПК) |
| `/chistory` | Показать историю команд |
| `/cclear` | Очистить историю команд |
| `/root del user` | Удалить пользователя (только для администратора) |
| `/root watch` | Просмотр всех пользователей (только для администратора) |
| `/root ncount root` | Перестать быть администратором (навсегда) |
| `/coin` | Орёл или решка (игра) |

> **Если что старые команды также есть в терминале!**

- **Также для безопасности был добавлен класс `CriticalPath` с `критическими путями, которые нельзя удалять` для `deldir <-rm/-rmtree>`**

- Для работы с командой `/screen` нужен файл `screen.py` - он уже добавлен в папку с программой `(работает только на Windows - так как использует модули win32)`

- Создается папка `chistory` - в ней файл `history.txt` (сохраняет историю команд) - также можно его удлаить через команду `cclear`, а посмотреть историю - `chistory`

#### Новая функция 14 `Математика AI`

- Новая функция для `Математики с AI (Elliot Bot Robot AI на базе DeepSeek-r1:8b)`, где можно написать выражение или вопрос по математике , а AI ответет на твой вопрос в скором времени

- `Не имеет памяти (сохранение чатов)`, а просто отвечает на вопросы

> **Функция 1 - Математика, также будет существовать и работать (может даже будет обновляться...)**


#### Новая функция 15 `Python помощь с AI`

- Новая функция для `Python помощь с AI (Elliot Bot Robot AI на базе DeepSeek-r1:8b)`, для работы с языком программирвоания `Python` (скоро может и другие...) или помощь с кодом, а AI ответет и поможет в скором времени

- `Не имеет памяти (сохранение чатов)`, а прсото помагает и объясняет

> **Функция 2 - Python помощь, также будет существовать и работать (может даже будет обновляться...)**
