## Требования для движка `AIEngine`:

### Конфиг

- В настройках включить поддержку ИИ

### Ollama

- [Сайт Ollama](https://ollama.com) 

- [Скачать Ollama](https://ollama.com/download/windows) или irm https://ollama.com/install.ps1 | iex (PowerShell)

- Затем: 

```bash
ollama pull deepseek-r1:8b
```

- Скачав сделайте проверку:

```bash
ollama serve    # Запуск Ollama в фоне
ollama run deepseek-r1:8b "Привет"   # Должно ответить
```

- `После в терминале скачать:`

```bash
pip install requests # Для сетевых запросов (Ollama)
```

- Если произошла ошибка: `Попробуйте снова`


## Требования для криптографического движка `CryptoElliot`:

- `В терминале скачать:`

```bash
pip install cryptography # Для Fernet, RSA, Padding, Serialization, Hashes, Default_Backend
```
