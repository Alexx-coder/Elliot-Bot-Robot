# Elliot Bot Robot v3.4.robot

Elliot Bot Robot - Новое поколение Elliot Bot, которое я переписал и добавил много фишек

## Прошлая версия (Прошлое поколение)

[Прошлая версия v3.3](https://github.com/Alexx-coder/Elliot-Bot-v3.3.git)

## Что добавлено в Elliot Bot Robot

### Функции

- Elliot Bot Robot AI - Не настоящий ИИ, который умеет отвечать на определённые команды, которые задал разработчик

- a1 - Функция доступная для админов (В разработке)

- u1 - Выход из аккаунта

- u2 - Завершение работы программы

### Подфункции

- Проверка сравнений в функции "Математика"

### Безопасность

- Теперь хэширование со солью

- Блокировка, если было много неудачных попыток входа (Решение в "Примечания")

- Логирование записей внутри Базы данных

- Защита от брутфорса (Итерации)

### База данных

- Раньше была JSON, теперь SQLite3

#### Таблицы для Базы данных

- Таблица users - Отвечает за хранение данных о пользователе

- Таблица messages - Отвечает за хранение сообщений (Вроде бы не используется)

- Таблица security_logs - Отвечает за хранение логирования 


### Структура программы

- Раньше ООП была слегка затронута, теперь она основа программы

### Доступ к админу

- Специальный код, который даст возможность стать админом и получить функции админа
