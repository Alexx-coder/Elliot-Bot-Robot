# Elliot Bot Robot

**Elliot Bot Robot** - `Новое поколение Elliot Bot`, которое я переписал и добавил много фишек

## Что нового добавлено в Elliot Bot Robot 3.6.robot

### Настройки (Функция 7):

- Добавлены новая подподфункция в `Система` 

- Добавлены подподподфункции в `Системных параметрах (Система)`: Права автовхода, Стать Администратором, Удалить аккаунт, Поддержка Elliot Bot Robot AI

### Конфиг (.ini)

- При включении программы будет создан файл `.ini` - там `Системные параметры (Аутенфикация, Настройки, Логирование)`

- Доступен автовход `(Изменить параметры можно в Настройках)`

### Логирование (.log)

- При включение программы будет создан файл `.log` - там находятся `шаги пользователей` в данной программе

- Уровень логирования: `INFO`

- Что записывается: `Время, Что произошло и Пользователь`

### Новая версия Elliot Bot Robot Ai v1.1 и Появление Движка AIEngine

- Добавлен Движок `AIEngine`

- Возможность обучать самостоятельно `через команды Elliot Bot Robot AI` с помощью Движка `AIEngine`

### Новый класс ошибок

- IncorrectAdminCode - `Неправильный код Администратора`

### Оптимизация

- Больше обработки ошибок `ElliotBotRobot Error`


### Безопасность

- Автовход происходит через захешированный используя соль `Токен Автовхода`

- Переход с обычных ID - `На uuid`

### Примечания

- `Конфиг и Логирование` хранятся `локально` на вашем устройстве 

- Если на ваш аккаунт привязана почта, забыв её название - вы не сможете войти в свой аккаунт. `Будьте осторожны!`

- Elliot_pw_worst: `Слабый пароль `

- Elliot_pw_middle: `Средний пароль`

- Elliot_pw_normal: `Рекомендуемый пароль`
