# Elliot Bot Robot v3.5.robot

Elliot Bot Robot - Новое поколение Elliot Bot, которое я переписал и добавил много фишек

## Что нового добавлено в Elliot Bot Robot 3.5.robot

### Переделанный функции:

#### Настройки (Функция 7):

- Добавлено 3 Подфункции (Для обычных пользователей) или 4 Подфункции (Для Администраторов)

- Подфункции: Профиль, Безопасность, Система, Администраторы, База Администратора (Доступно только Администраторам)

- Подподфункции: Поменять имя (Профиль); Поменять пароль, Подключить почту, Поменять почту, Моя почта (Безопасность); Общая информация (Система); Все Администраторы (Администраторы); Удаление пользователя, Просмотр всех пользователей, Перестать быть Администратором (База Администратора)


### Что убрано

- Функции для Администраторов в меню: Перенесены в Базу Администратора (Настройки)

### Аутенфикация

- Выбор входа как Призрак: Без сохранений в Базе данных, Любое имя (Не логин), Не нужны пароли

- Если указана почта в Настройках (Безопасность) - При входе в аккаунт попросят вашу почту (Название вашей почты, Например - ваша.почта@сервис.домен)

### Исправление багов

- Исправлен баг связанный с блокировкой на любые аккаунты, после того как получили временную блокировку на вход аккаунт (на который пытались зайти, но произошли неудачные попытки)

## Примечания

 - Если на ваш аккаунт привязана почта, забыв её название - вы не сможете войти в свой аккаунт. Будьте осторожны!

- Elliot_pw_worst: Слабый пароль 

- Elliot_pw_middle: Средний пароль

- Elliot_pw_normal: Рекомендуемый пароль
