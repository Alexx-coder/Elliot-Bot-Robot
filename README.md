## **Как поставить Ollama для AI (DeepSeek-r1:8b)**

> **Инструкиця для установки:**

- [Сайт Ollama](https://ollama.com) 

- [Скачать Ollama](https://ollama.com/download/windows) или 
 
```powershell
irm https://ollama.com/install.ps1 | iex (PowerShell)
```

- Затем: 

```bash
ollama pull deepseek-r1:8b
```

- Скачав сделайте проверку:

```bash
ollama serve    # Запуск Ollama в фоне
ollama run deepseek-r1:8b "Привет"   # Должно ответить
```

- `После в терминале скачать:`

```bash
pip install requests # Для сетевых запросов (Ollama)
```

- Если произошла ошибка: `Попробуйте снова`

## **Предупреждения**

> **Для качественной работы Elliot Bot Robot вы должны одходить к требованиям:**

1. **Минимальная версия Python: 3.8 | Оптимально: 3.10 | Рекомендуется: 3.13 (самая лучшая)**

2. **Для AI обратитесь в прошлый раздел (если не установлено)**

3. **Операционная система: Windows 10, 11 (необязательно, но мелких операций)**

4. **Установить бибилиотеки (или использовать инструкцию в следующем разделе) :**

```text
cryptography
requests
psutil
python-whois
pywin32
```

5. **Для быстрой установки иметь `Git` (необязательно)**

6. **Для функции с скриншотом иметь файл `screen.py` в той же папке где проект**

> **Остальные предупрждения связанные с работой**

- **Не пугайтесь если создались папки: `keys`, `chistory`, `notes`, `__pycache__`**
