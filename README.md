# Elliot Bot Robot

Elliot Bot Robot - Консольный бот, новое поколение Elliot Bot

## Что себя представляет

- Консольный бот, с разными и полезными функциями 

- Имеет безопасное и локальное хранение данных о пользователях 

### Некоторые функции

- Проверка валидации Email и Номеров телефонов из РФ (Проверка по паттерну)

- Генератор пароля

- Математические операции

- Параметры

### Что может появиться в будущем

- Новые функции

- Более улучшенная безопасность данных

- Больше оптимизации

- Вэб-интерфейс (LocalHost)
